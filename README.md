# ctetris

cursed CUI tetris for minimalists.

## Build

ncurses is required.

```
$ make
```

## Run

```
$ ./tetris
```

## Controller

|  key | |
|---|---|
|  ← →| move left/right |
| ↑ | hard drop |
| ↓ | soft drop |
| z | rotate left |
| x | rotate right |
| c | hold |
| p | pause game |
| q | quit game |
